## Bug report model 

Bug reports should be made with the following form : 

### Title : Should be short and clear 

- Bug ID (refer to bug table)
- Statut 
- Severity
- Linked test cases (refer to test cases ID)
- Linked risk (refer to risk analysis ID)
- Description
- Steps to reproduce 
- Actual result 
- Expected result
